# Documentacion oficial de Git y GitHub

Este curso mejora mucho si acompanias la practica con las referencias
oficiales de Git y GitHub.

## Referencias principales

- [Git documentation](https://git-scm.com/doc)
- [Pro Git book](https://git-scm.com/book/en/v2)
- [GitHub Docs](https://docs.github.com/)

## Recursos clave para estudiar

- [Referencia de comandos Git](https://git-scm.com/docs)
- [Configurar Git por primera vez](https://git-scm.com/book/en/v2/Getting-Started-First-Time-Git-Setup)
- [Git branching](https://git-scm.com/book/en/v2/Git-Branching-Branches-in-a-Nutshell)
- [Pull requests en GitHub](https://docs.github.com/en/pull-requests)
- [GitHub Actions](https://docs.github.com/en/actions)
- [GitHub Markdown](https://docs.github.com/en/get-started/writing-on-github)

## Orden recomendado

1. Aprende Git local primero.
2. Luego practica ramas y merges.
3. Despues conecta el flujo con GitHub.
4. Finalmente estudia PRs, Actions y colaboracion.
